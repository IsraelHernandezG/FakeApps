Fake App DataBase

Una base de datos abierta de aplicaciones falsas para todo tipo de tareas y proyectos escolares, libre de derechos para ser usadas en cualquier lugar y ocasión, adem{as se invita a la comunidad para que participe en este proyecto y haga crecer la base de datos.

La base de datos esta conformada por 5 campos: el nombre de la Aplicación, el nombre del Desarrollador, la fecha en que se liberó, la compañía creadora o patrocinadora y la ultima versión liberada.

Elementos:
    <% apps.forEach(function(app) { %>

    ID: <%= app.id %>
    Nombre: <%= app.name %>
    Desarrollador: <%= app.developer %>
    Fecha de liberación: <%= app.releasedate %>
    Compañía: <%= app.company %>
    Versión: <%= app.version %>

    <% }); %>
